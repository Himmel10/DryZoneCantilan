# Dry Zone - Cantilan

A modern, responsive web application for local laundry shop directory and service ordering in Cantilan, Surigao del Sur. Built on a **circular economy model** where all participants benefit.

## Features

- **Shop Directory**: Browse and search for laundry shops in the area
- **Service Booking**: View services and add them to cart
- **Pickup & Delivery**: Convenient logistics service with integrated tracking
- **User Authentication**: Customer and seller account management
- **Admin Dashboard**: System maintenance and monitoring
- **Service Provider Dashboard**: Seller shop management
- **Chat System**: Real-time communication between customers and shops
- **Order Management**: Track orders and manage service requests
- **Reviews & Ratings**: Customer feedback system
- **Responsive Design**: Mobile-friendly interface

## Technology Stack

- **Backend**: PHP (MySQL)
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: MySQL
- **Server**: Apache (XAMPP)

## Installation

1. Clone the repository
2. Copy to your XAMPP htdocs folder
3. Create database using provided schema
4. Configure `db.php` with your database credentials
5. Access via `http://localhost/DryZoneCantilan`

## User Roles

- **Customer**: Browse shops, book services, track orders, use pickup/delivery
- **Seller (Service Provider)**: Manage shop details, view orders, update status, grow business
- **Admin**: Manage system, monitor users, view statistics, optimize operations

## Revenue Streams

1. **Pickup & Delivery Fees**: Primary income source
2. **Commission on Orders** (Optional): Small percentage from transactions
3. **Premium Shop Listings** (Future): Featured shop placements
4. **Analytics & Business Insights** (Future): Sell data to service providers
